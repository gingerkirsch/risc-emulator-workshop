• 00 - "Empty" - moves instruction pointer without side effects
  • 11 - "Read" - reads the value from input
  • 11 - "Write" - prints the value
  • 20 - "Load" - loads the value into register
  • 21 - "Store" - stores the value from the register in the memory
  • 30 - "Add" - addition of values in the memory and in the register
  • 31 - "Sub" - subtraction of value in the memory from the value in the register
  • 32 - "Div" - division of value in the register by value from memory
  • 33 - "Mul" - multiplication of values in the memory and in the register
  • 40 - "Go" - jump to particular memory cell
  • 41 - "GoNeg" - jump to particular memory cell if value in register is negative
  • 42 - "GoZero" - jump to particular memory cell if value in register is zero
  • 43 - "Halt" - stop the execution
